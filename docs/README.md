# Vibe Hub — Documentação do Projeto

## O que é o Vibe Hub?

O **Vibe Hub** é um site de curadoria que apresenta as principais ferramentas, frameworks e tecnologias que estão moldando o futuro da engenharia de software e inteligência artificial em 2026.

O projeto adota uma estética brutalista elegante, com efeitos visuais sofisticados (como o prisma Animated Background), transmitindo a essência do "vibe coding" — uma abordagem moderna e descontraída ao desenvolvimento.

## Propósito

O Vibe Hub serve como **guia de referência rápida** para desenvolvedores que desejam:

- Descobrir as ferramentas mais relevantes do ecossistema atual
- Comparar opções em diferentes categorias (frontend, backend, banco de dados, IA, etc.)
- Acessar links diretos para documentação e repositórios oficiais
- Manter-se atualizado sobre as tendências do mercado

## Stack Técnica

- **Frontend**: React + TypeScript + Vite
- **Estilização**: TailwindCSS
- **Animações**: Framer Motion
- **Ícones**: Lucide React
- **Fonts**: Geist (sans-serif) + Mono

## Como Executar

```bash
cd vibe-hub
npm install
npm run dev
```
